# Avalon ERP Platform

Avalon — это современная ERP-платформа, созданная для быстрого моделирования бизнес-процессов и автоматизаций без программирования.  
Платформа поддерживает SaaS-развертывание и локальную установку для корпоративных нужд.

## Основные возможности
- **Мультитеннантность** — безопасная изоляция данных организаций.
- **Метамоделирование** — создание сущностей, связей и форм без изменения кода.
- **AI-помощник** — генерирует модели данных, справочники и процессы по запросу.
- **Готовая инфраструктура** — Postgres, Redis, MinIO, Nginx, Docker.
- **React + Vite frontend** и **Node.js / TypeScript backend**.

Avalon позволяет создавать ERP-решения и автоматизации под любую компанию: CRM, склад, документы, регистры, производство и многое другое.
