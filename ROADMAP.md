# Avalon — Roadmap

## Готово
- Docker-инфраструктура
- Backend/Frontend структура
- Мультитеннантность (базовая)
- Интеграции: Redis, Postgres, MinIO
- React + Vite фронт
- Nginx reverse proxy

---

## В разработке
- Метамодель сущностей  
- Генератор форм UI  
- Система workflow (статусы, переходы)  
- Система прав доступа  
- AI-помощник (v1)  
- CRUD-генерация по метаданным  

---

## Планируется
### AI
- Автоматическая генерация сущностей
- Генерация процессов

- Модули: CRM, склад, документы, финансы  
- Интеграции: Telegram, Email, Webhooks  
- Версионирование метаданных  
- Конструктор UI-виджетов  

### SaaS
- биллинг  
- лимиты  
- управление тенантами
